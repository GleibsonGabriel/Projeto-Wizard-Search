WIZARD SEARCH
Plataforma Conceitual de Automação em Prevenção à Fraude

O Wizard Search é um sistema conceitual desenvolvido
como projeto acadêmico voltado à automação de análises
em prevenção à fraude, onboarding bancário e processos
de KYC (Know Your Customer).

A proposta do sistema consiste em consumir dados
estruturados provenientes de APIs proprietárias,
arquivos JSON e fontes de inteligência aberta (OSINT),
centralizando informações financeiras, cadastrais
e transacionais em uma única interface visual.

Com base nas variáveis identificadas, o sistema
categoriza eventos em níveis de criticidade:

- Informações Subjetivas
- Informações Suspeitas
- Informações Críticas

Cada variável recebe um peso dentro de um sistema
de score automatizado, permitindo que o analista
priorize investigações e interprete riscos
de maneira rápida, visual e intuitiva.

O protótipo apresentado neste projeto simula:

- Consumo de APIs
- Consolidação de dados AML
- Interpretação de variáveis de fraude
- Sistema de score
- Semáforo de risco
- Simulação de ingestão JSON
- Dashboard analítico corporativo

Objetivo acadêmico:
Demonstrar conceitos de automação,
tratamento de dados, UX aplicada
à prevenção à fraude e categorização
de risco operacional.

Tecnologias utilizadas:
- React
- Bootstrap
- JavaScript
- Vite
